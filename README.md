## The Cost Analysis (TL;DR)

Every AI coding tool is being subsidized right now. Here's what I found:

- **Claude Code** — $200/mo subscription can deliver $3,650+/mo in API-equivalent usage (18x subsidy)
- **Codex** — OpenAI admits they lose money on $200/mo Pro subscribers. 4x more token-efficient than Claude Code per task.
- **Gemini** — Cheapest API prices by far (10x less than Claude). Google cross-subsidizes with $175B in ad revenue.
- **Perplexity Computer** — Pays API prices to ALL of these providers as a middleman, passes costs to you as credits.
